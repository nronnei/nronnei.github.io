Apps

A few examples of the things I like to build

Bike Share Explorer

Personal Project

A cool little app that uses the Generalized Bikeshare Feed Specification (GBFS) to explore bike share stations around the world. Created using Mapbox GL JS and Google’s Material Design Light, this map leverages the beautiful cartography and cutting edge mapping tech of Mapbox along with the clean, simple, and beautiful look of MDL for a responsive web map that looks great.

Check it out



Transit to Green Space Project

North Star Sierra Club

This app helps people get to public parks using public transit. I developed it for the North Star Sierra Club in 2013. It uses a responsive jQuery sidebar and the Google Maps JavaScript API for a simple interface that looks good on just about any platform.

Check it out or see the code onGithub.
